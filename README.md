<h1>幹事サポート</h1>
<p>どういうサービス？：１０名以上で遊べるスポットを探せるサイト</p>
<p>何ができるの？：GoogleMapを見ながら、スポットのレビューを投稿・確認</p>
<p>アピールポイント：GoogleMapApiを入れているため、視覚的に情報収集可能</p>
